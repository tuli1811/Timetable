## ğŸ“ Description

The **Study Timetable Web Application** is developed using **HTML, CSS, and JavaScript**, with **Firebase Email/Password Authentication** used as the backend for secure user login and registration.

When a user visits the application URL, they can either create a new account or log in using their existing email and password. After successful authentication, users can create and manage their study timetable by selecting days, subjects, time slots, and tasks. The application also allows users to edit or delete tasks and download the complete weekly routine as a PDF file using **jsPDF**.
